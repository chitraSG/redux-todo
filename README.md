## Cookpal (redux-toolkit)

Recipe browse + cart demo built with React 19 and Vite.

### Stack
- React, Vite, React Router, Redux Toolkit, Tailwind, EmailJS, Swiper

### State
- **Redux:** cart (items, quantities)
- **React state:** filters, pagination on Explore
- **Data:** static `src/data/recipes.js`
- **Persistence:** localStorage (`cookpal-cart`)

### Not in this project
- Backend API, auth, database, TypeScript, RTK Query
- Redux for filters (by design)
